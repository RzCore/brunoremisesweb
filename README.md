# 🚗 Remises Bruno - Sitio Web Moderno

## 📋 Descripción

Sitio web completamente modernizado para **Remises Bruno**, una empresa de transporte con más de 25 años de experiencia en Chivilcoy, Buenos Aires. El sitio web presenta un diseño contemporáneo, responsive y optimizado para ofrecer la mejor experiencia de usuario.

## ✨ Características Principales

### 🎨 Diseño Moderno
- **Diseño responsive** que se adapta a todos los dispositivos
- **Paleta de colores moderna** con gradientes y efectos visuales
- **Tipografía optimizada** usando Inter y Poppins
- **Animaciones suaves** y transiciones fluidas
- **Efectos de hover** interactivos

### 🚀 Funcionalidades Avanzadas

## 🔧 Instalación y Uso

### Requisitos
- Navegador moderno (Chrome, Firefox, Safari, Edge)
- Servidor web (opcional para desarrollo local)

### Instalación
1. Clona o descarga el proyecto
2. Abre `index.html` en tu navegador
3. Para desarrollo, usa un servidor local

### Personalización
1. **Colores**: Modifica las variables CSS en `:root`
2. **Contenido**: Edita el HTML según necesidades
3. **Imágenes**: Reemplaza en `assets/img/`
4. **Funcionalidades**: Modifica `assets/js/main.js`
